# ASEE Flow Quick Start - Get Started with ASEE Flow

This repository contains the BPMN process and DMN decision table as well as the example code for the get started guide at [https://docs.aseeflow.org/get-started/quick-start/](https://docs.aseeflow.org/get-started/quick-start/).

Every step of the tutorial was tagged in this repository. You can jump to the final state of each step by the following command:

```
git checkout -f Step-X
```
